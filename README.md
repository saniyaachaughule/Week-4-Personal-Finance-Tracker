# 💰 Personal Finance Tracker

A Python-based Personal Finance Tracker that helps users manage daily expenses, organize spending by category, generate reports, set budgets, and save data using JSON and CSV files.

This project was developed as the **Week 4 Final Project** for The Developers Arena Python Internship.

---

## 📌 Project Objectives

- Manage personal expenses efficiently
- Store expense data permanently using JSON
- Generate monthly and category-wise reports
- Track budgets and spending
- Practice Python file handling, modular programming, and error handling

---

## 🚀 Features

- ✅ Add new expenses
- ✅ View all expenses
- ✅ Search expenses
- ✅ Delete expenses
- ✅ Monthly expense report
- ✅ Category-wise report
- ✅ Expense statistics
- ✅ Text-based expense chart
- ✅ Budget management
- ✅ Export expenses to CSV
- ✅ Backup expense data
- ✅ Restore backup data
- ✅ JSON file storage
- ✅ Input validation
- ✅ Exception handling
- ✅ Menu-driven interface

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/saniyaachaughule/week4-finance-tracker.git
```

### Go to project folder

```bash
cd week4-finance-tracker
```

### Run the application

```bash
python run.py
```
